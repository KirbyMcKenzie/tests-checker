# Tests Checker

> A GitHub App built with [Probot](https://github.com/probot/probot) that require writing the tests in the Pull Requests.

## Setup

```sh
# Install dependencies
npm install

# Run typescript
npm run build

# Run the bot
npm start

# or for development
npm run dev
```
